# 闲鱼捡漏助手

一个手机友好的静态 PWA，用来评估闲鱼二手商品是否值得捡漏。它不会抓取平台数据，而是基于用户录入的信息判断价格优势、成色可信度、卖家风险和交易安全。

## 功能

- 输入商品价格、市场参考价、成色、凭证和卖家描述
- 自动计算 0-100 分捡漏评分
- 识别疑似二道贩子/职业卖家信号
- 输出建议出价区间、行动建议和风险提示
- 本地自动保存，支持手机浏览器添加到桌面
- 纯静态文件，无需后端、无需外部 CDN

## 本地预览

直接打开 `index.html` 即可使用。部署到 GitHub Pages、Cloudflare Pages、Netlify 或国内静态托管也不需要改代码。

## 评分逻辑

综合分由四类信号加权而来：

- 价格：闲鱼价格相比市场参考价的折扣幅度
- 品质：成色、购买凭证、描述里的正向/风险关键词
- 卖家：自用、一手、实拍、验货支持、批量卖货迹象
- 交易：同城面交、平台担保、不跳平台、不付定金

这个工具只做辅助判断，不能替代实物验货和平台交易保障。
