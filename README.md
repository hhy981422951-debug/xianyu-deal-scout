# 闲鱼捡漏助手

一个手机友好的静态 PWA。用户只需要粘贴闲鱼商品链接，页面会识别链接来源和商品 ID，并生成风险判断、验货清单、卖家沟通话术和最近分析记录。

## 功能

- 只输入一个商品链接，不再要求填写价格、成色、凭证等复杂信息
- 自动识别闲鱼/淘系链接、短链和可能的商品 ID
- 生成卖家风险提醒、验货步骤和可复制沟通话术
- 本地保存最近 5 条分析记录
- 手机响应式界面，支持 GitHub Pages 静态部署
- 无后端、无外部 CDN、无平台抓取

## 说明

纯静态网页不能绕过闲鱼登录、风控或跨域限制抓取真实商品价格和卖家主页。本工具的定位是“链接核验工作流”：让用户从链接开始，快速知道该问什么、该看什么、哪些信号要警惕。
